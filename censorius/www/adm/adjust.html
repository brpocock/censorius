Enter an adjustment

To enter an adjustment to an FPG invoice, you'll need:

  • The invoice number
  • The admin key

The admin key is displayed when you call up the guests' invoice using the link that was mailed to you, or a search form. If you enter it incorrectly, nothing good will happen.
